圧縮空気分析計 (CAA)

記録の測量値を使って圧縮空気システムを分析するアプリケーション

圧縮空気分析計（CAAを略称する）は CSSoft 又は CS-Bus データベースに記憶された測量値を使って圧縮空気システムを分析するアプリケーションです。このプログラムを利用して、グラフや統計の形式で測量値を調べることができて、それによって圧縮空気システムを全面的に理解します。

CAA は下記の特性を提供しました。

  • 異なったバージョン又はコピーの CSSoft や CS-Bus で生成した多数のデータベースを管理します

  • 一回に一つのデータベースを接続して、その中からすべての記録ファイルを取り出します

  • グラフで単独や多数の記録ファイルに記憶された測量値を表示します。

  • グラフビューをカスタマイズします

  • リーク設定と会社の情報をカスタマイズします

  • ユーザーの定義された設定によって関連測量値の統計データを生成します

  • 一部の会社情報、コメント、グラフと統計データを含んだレポートを印刷します

  • レポートの全部又は一部をJPEG や PDF ファイルへエックスポートします

  • 記録ファイルを現在開けられたデータベースにエックスポートします

  • データベースの中の記録ファイルを一つの CS データファイルへエックスポートします

  • レポートを .rep ファイルに保存して、そして将来にロードします

  • 同じなグラフで異なった日数と週間数の測量値を調べます
