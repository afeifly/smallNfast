データベースの配置と記録ファイルのビュー

プログラムを起動した後に、 CAA がアクセスするデータベースを配置する必要がある可能性があります。このために、メニューの「File -> Select Database（ファイル -> データベースの選択）」を選択して、「Select Databaseダイアログボックス」を開けます。このダイアログボックスを利用して、使用可能のデータベースを調べて、開ける必要があるデータベースを選んだり、データベースを追加、編集、削除したりすることができます。

一つのデータベースを開けた後に、記憶された一つ又は多数の記録ファイルを選択して、グラフエリアで調べることができます。このために、ツールバーにあるファイル選択ボタン(  )をクリックして「Select File ダイアログボックス」を開けます。このダイアログボックスを利用して、このデータベースに記憶されたすべての記録ファイルを調べて、一つ又は多数のファイルを選んで調べることができます。グラフエリアの特性の詳しい情報に関して、「グラフエリア」を参照してください。
