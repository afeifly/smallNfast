グラフと統計レポートのエックスポート

現在に表示されたグラフと/或は統計レポートを JPEG や PDF ファイルへエックスポートすることができます。エックスポート操作を必要とする場合、ツールバーにあるボタン（ ）をクリックしてください。この時に「Export to JPEG/PDF File ダイアログボックス」が表示されます。



Export to JPEG/PDF File(s) dialog


上記のダイアログボックスの中で、グラフと/或は統計レポートを JPEG や PDF ファイルへエックスポートすることを選択することができます。ダイアログボックスの中の一つのボタンをクリックすると、次の通りの別のダイアログボックスが表示されて、ここでファイルを保存することができます。



Dialog to save the file


グラフと統計レポートを JPEG ファイルへエックスポートすることを選択すると、プログラムは単独で二つのファイルを保存することを提示しますが、一つのファイルがグラフの保存に用いられて、もう一つが統計レポートの保存に用いられます。その他のオプションを選択すると、プログラムは一つのファイルしか保存する必要がないと提示します。プログラムは選択された最後の保存ファイルの位置を記憶します。
