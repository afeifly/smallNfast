レポートを JPEG やPDF ファイルへエックスポートします

レポートの全部又は一部をJPEG や PDF ファイルへエックスポートすることができます。この機能を利用して、レポートの電子コピーを新規作成することができて、それによって他人と共有したり、ネット上で転送します。エックスポート操作を必要とする場合、メニューの「File -> Export To PDF（ファイル ->  PDF ファイルへのエックスポート）」を選択することができます。この時に「Export to JPEG/PDF File（s） ダイアログボックス」が表示されます。



Export to JPEG/PDF File(s) ダイアログボックス


このダイアログボックスで、下記のことを選択することができます。

  • メイングラフと/或は主統計レポートを JPEG ファイルへエックスポートすること。

  • メイングラフ、主統計レポート、標準フルレポート（又はフルレポート）又は詳しいフルレポートを一つの PDF ファイルへエックスポートすること。現在開けられた記録ファイルの生成時間が１日間を上回った場合だけ、詳しいフルレポートを選択することができます。詳しいレポートは標準レポートのすべてのページを含んで、また毎日のグラフと統計データを含んでいます。

エックスポートを必要とする場合、必要のオプションの側のラジオボタンをクリックして、「Export」ボタンをクリックしてください。この時に次の通りの別のダイアログボックスを表示して、その中でファイルを保存することができます。



Dialog to save the file


グラフと統計レポートを JPEG ファイルへエックスポートすることを選択すると、プログラムは単独で二つのファイルを保存することを提示しますが、一つのファイルがグラフの保存に用いられて、もう一つが統計レポートの保存に用いられます。その他のオプションを選択すると、プログラムは一つのファイルしか保存する必要がないと提示します。プログラムは選択された最後の保存ファイルの位置を記憶します。
