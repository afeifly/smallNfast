記録ファイルのインポート

一つのCS データファイル（.csd）に含まれた記録ファイルを現在に開けられたデータベースにインポートすることができます。CS データファイルは CSSoft や CS-Bus で生成されて、一つ又は多数の記録ファイルを含んでいます。この機能を通して、その他のコンピュータが入手した測量値を調べることができます。

インポートを必要とする場合、メニューの「File -> Import Data（ファイル ->  データのインポート）」を選択してください。この時に「Import ダイアログボックス」が表示されます。



Import dialog


わりに大きいファイルをインポートする場合、何分間を必要とする可能性があります。プログラムはインポート結果を表示します。成功にインポートした後に、すべての当該 CS データファイルに含まれた記録ファイルがインポートされて、そして「Select Record File ダイアログボックス」に表示されます。
