メインウィンドウの概述

下の図には典型的なCAA メインウィンドウを表示します。



A typical CAA main window


  • メニューバーは五つのメニューを含んで、異なっている操作を行うことができます。これらのメニューの詳しい説明は「メニュー」を参照して下さい。

  • ツールバーの大多数の機能はメニューの機能と同じですが、これらのボタンはアクセスプログラムの基本的な特性のショートカットを提供しました。あるボタンの機能を調べようとする場合、マウスをその上に置いてください。しばらくすると、当該ボタンの機能を説明する簡潔なヘルプ情報が表示されます。これらのツールバーのボタンの詳しい説明は「ツールバー」を参照してください。

  • グラフエリアは現在開けられた記録ファイルのグラフを表示します。グラフエリアは次の通りの特性を持っています。

    • グラフエリアは一本の横軸（時間軸）と多とも4本の縦軸を備えています。目盛り単位は各縦軸の側に表示されています。

    • グラフにある各曲線はある特定のチャンネルの測量値を代表して、その色がチャンネル名の色と同じです。グラフには多とも 6 本のチャンネルの測量値を表示することができますが、これらのチャンネルの異なる単位の数量が四つを上回ってはいけません 。「Select Channel ダイアログボックス」の中でツールバーにあるチャンネル選択ボタン（  ）をクリックしてグラフの中のチャンネル表示を変更することができます。

    • ツールバーにある文字変更ボタン（  ）をクリックしたり、メニューの「Setting -> Change Text（設定 -> 文字変更）」を選択したりして、タイトル、軸名称と凡例（即ちチャンネル名と単位）を含め、グラフの文字表示をカスタマイズします。凡例を除き、すべての入力文字が保存されて、そして次回のプログラムのスタート時にロードされます。



      Change text ダイアログボックス


    • 記録ファイルの周期が長すぎると、右下の「--->」や「<---」のボタンをクリックしてグラフの異なる部分を調べることができます。

    • ツールバーにあるレポートタイプ選択ボタン（ ）をクリックしたり、メニューの「View -> Select Report Type（ビュー -> レポートタイプ選択）」を選択して、「Select Report Type ダイアログボックス」を開けたりして，一日間、一週間又は一ヶ月間のグラフを調べることができます。

      デフォルトで、各可視チャンネルのすべての測量値がグラフエリアに表示されます。比較的に短い周期で多すぎる値を表示した場合、曲線ははっきりしなくなる恐れがあります。更にはっきりに曲線を調べようとする場合、このダイアログボックスで1分間、15分間又は1時間あたりの各チャンネルの平均測量値の計算を選択することができて、表示された値の数量を減らします。



      Select report type ダイアログボックス


    • ツール・バーにある時間周期選択ボタン（ ）をクリックしたり、メニューの「View -> Select Time Period（ビュー -> 時間周期選択）」を選択したりして、グラフに表示される測量値の周期を指定します。Time（時間）軸の目盛りは指定された周期によって変化します。グラフを原状に回復する場合、メニューの「 View -> Reload（ビュー -> リロード）」を選択してください。



      Time option ダイアログボックス


    • ツールバーにあるコメント追加ボタン（ ）をクリックしたり、メニューの「Tool -> Add Comment To Graph（ツール -> グラフコメントの追加）」を選択したりして，グラフにコメントを追加します。コメントを追加した後に、それをグラフのいかなる位置にドラッグ・アンド・ドロップすることができます。また、コメントを編集したり削除したりすることもできます。コメントを右クリックして、相応のメニューを選択するとOKです。追加したコメントがグラフの印刷とエックスポート時にも表示されます。

    • グラフエリアを右クリックして、「Reload」を選択したり、メニューの「View -> Reload（ビュー -> リロード）」を選択したりして、グラフにある測量値の周期を表示することできます。
