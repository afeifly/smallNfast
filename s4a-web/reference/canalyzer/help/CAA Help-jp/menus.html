メニュー

Fileメニュー

  • Select File（ファイルの選択） － 現在開けられたデータベースに記憶されたすべての記録ファイルを表示して、一つ又は多数のファイルを選択して、グラフで調べることができます。もっと多い情報は、「Select Record File ダイアログボックス」を参照してください。

  • Select Database（データベースの選択） － すべての使用可能のデータベースを表示して、一つを選択して開けることができます。もっと多い情報は、「Select Database ダイアログボックス」を参照してください。

  • Save Report（レポートの保存） － 現在開けられた記録ファイルに関連する情報を .rep ファイルの中に保存します。もっと多い情報は、「レポートの保存とロード」を参照してください。

  • Load Report（レポートのロード） － 一つの.rep ファイルから記録ファイルに関連する情報をロードします。もっと多い情報は、「レポートの保存とロード」を参照してください。

  • Import Data（データのインポート） － 一つの CS データファイル（.csd）を現在開けられたデータベースにインポートします。もっと多い情報は、「記録ファイルのインポート」を参照してください。

  • Export To PDF（PDFファイルへのエックスポート） － 生成したレポートの全部又は一部を JPEG や PDF ファイルへエックスポートします。もっと多い情報は、「レポートを JPEG や PDF ファイルへエックスポート」を参照してください。

  • Page Setup（ページの設定） － レポートのページ印刷オプションを選択します。もっと多い情報は、「ページ印刷オプションの選択」を参照してください。

  • Print Preview（印刷プレビュー） － レポートのプレビューと印刷。もっと多い情報は、「レポートのプレビューと印刷」を参照してください。

  • Print（印刷） － レポートのプレビューと印刷。もっと多い情報は、「レポートのプレビューと印刷」を参照してください。

  • Exit（終了） － アプリケーションを終了します。終了の時に自動的にユーザーの設定を保存します。

Setting メニュー

  • Change Text（テキストの変更） － タイトル、軸名と凡例を含め、グラフに表示されるテキストを変更します。

  • Calculation Setting（計算の設定） － エネルギーと圧縮空気のコストの計算に用いられる設定を指定します。もっと多い情報は、「Calculation Setting ダイアログボックス」を参照してください。

  • Basic Setting（基本設定） － 会社の基本的な情報を入力します。もっと多い情報に関して、「基本的な情報の入力」を参照してください。

Tool メニュー

  • Show Statistics（統計データの表示） － 統計レポートを調べて、レポートには計算設定と測量データによって算出されたシステムの静態値を表示します。

  • Define Leakage（リークの定義） － グラフで一本のリーク線を表示して、システムの平均的なリーク量を定義するのに用いられます。当該メニューを再選択すると、当該リーク線を隠します。

  • Select Channel（チャンネルの選択） － グラフにあるチャンネルを選択して表示します。もっと多い情報は、「Select Channel ダイアログボックス」を参照してください。

  • Compare Data（データの比較） － 同じなグラフで異なった日数と週間数の測量値を調べます。詳しくは「同じなグラフで異なった日数と週間数の測量値を調べる」を参照してください。

  • Add Comment To Graph（グラフコメントの追加） － グラフでコメントを追加します。もっと多い情報は、「グラフコメントの追加」を参照してください。

  • Write Comment Page（コメントページの作成） － コメントページでコメントを入力して、これらのコメントが印刷/エックスポートされたレポートに含まれています。

Viewメニュー

  • Select Report Type（レポートタイプの選択） － 一日間、一週間又は一ヶ月間のグラフを選択して調べます。また、1分間、15分間又は1時間あたりの各チャンネルの平均的な測量値を選択することができて、表示される値の数量を減らします。

  • Select Time Period（時間周期の選択） － グラフにある測量値の周期を指定して表示します。もっと多い情報は、「時間周期の選択」を参照してください。

  • Reload（リロード） － 現在の記録ファイルをリロードして、そしてグラフを初期状態に回復します。

  • Display Records At Cursor（マウス位置の記録の表示） － マウス位置の時間と使用可能の測量値に対して動態表示機能を起動します。時間と測量値がグラフのトップに表示されます。このメニューを再選択すると、当該機能を停止します。

Helpメニュー

  • Help Contents（ヘルプの内容） － アプリケーションのヘルプ内容を表示します。

  • Register Software（ソフトウェアの登録） － ライセンス情報のビューとソフトウェアの登録。

  • About（アバウト） － バージョンの情報を表示します（ユーザーのサポートを必要とする場合、必ずバージョンの情報を提供しなければなりません）。
