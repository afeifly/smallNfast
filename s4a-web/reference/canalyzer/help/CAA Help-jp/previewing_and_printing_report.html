レポートのプレビューと印刷

CAAを利用して、現在開けられた記録ファイルに会社の情報、コメント、グラフと統計データを含んだレポートをプレビューして印刷することができます。メイングラフだけを印刷することもできます。

基本情報の入力

フルレポートを印刷する必要がある場合、「Basic Information ダイアログボックス」に会社の基本情報を入力して、この情報が印刷/エックスポートされたレポートの表紙に示されます。ツールバーにある基本設定ボタン（ ）をクリックしたり、メニューの「Setting ->Basic Setting（設定 -> 基本設定）」を選択したりすると、「Basic Information ダイアログボックス」を開けることができます。このダイアログボックスで、社名、住所、電話などを入力することができます。



Basic information ダイアログボックス


  • 会社のログを設定する場合、「Select Logo File」ボタンをクリックして、「Load ダイアログボックス」でログファイルを選択してください。このログが印刷レポートの各ページ右上に表示されます。



    Load ダイアログボックス


「OK」ボタンをクリックすると、すべての入力情報が保存されて、そして次回にプログラムを起動する時にロードします。

コメントの入力

レポートにコメントページを追加することを選択することができますが、コメントページは通常関連レポートの要旨や分析を含みます。コメントページは印刷/エックスポートされたレポートの第二ページに示されます。一枚のコメントページを追加する必要がある場合、メニューの「Tool -> Write Comment Page（ツール -> コメントページの作成）」を選択して、「Comment Pageダイアログボックス」を開けて、それからタイトルと内容を入力します。「OK」ボタンをクリックすると、すべての入力されたコメントが保存されて、そして次回にプログラムを起動する時にロードします。レポートからコメントページを削除しようとする場合、当該ダイアログボックスの「 Clear Comment」ボタンをクリックするだけでOKです。



Comment page ダイアログボックス


ページ印刷オプションの選択

レポートを正しくプレビューして印刷するために、「Page Setup ダイアログボックス」の中のページ印刷オプションを正しく選択しなければなりません。メニューの「File -> Page Setup（ファイル -> ページ設定）」を選択したり、「Print Previewダイアログボックス」の中の「Page Setup」ボタンをクリックしたりすると、「Print Preview ダイアログボックス」を開けることができます。



ページ設定ダイアログボックス（デフォルトのページ印刷オプションを表示します）


このダイアログボックスはこのグラフを印刷するプリンターのデフォルト/ユーザが定義したページ印刷オプションを表示します。ここで、使う必要がある紙の大きさ（プリンターに入れた紙とここに指定したものと同じなことを確保）とマージンを選択することができます。当該レポートの印刷用プリンターを変更しようとする場合、「 Printer...」ボタンをクリックして、それからプルダウンリストから別のプリンターを選択してください。「OK」ボタンをクリックすると、ここで選択されたオプションが保存されて、そして次回にプログラムを起動する時にロードします。

注：このダイアログボックスで、レポートの各ページの方向を変更することができません。グラフのページがいつも横方向で印刷されて、その他のページが縦方向で印刷されます。

レポートのプレビューと印刷

ツールバーにある印刷プレビューボタン（ ）をクリックしたり、メニューの「File -> Print Preview（ファイル -> 印刷プレビュー）」を選択したりして、「Print Preview ダイアログボックス」でレポートをプレビューします。



Print preview ダイアログボックス


このダイアログボックスを利用して、スクリーンでレポートの印刷状況をプレビューすることができます。また、このダイアログボックスはレポートの印刷用プリンターの名称も表示します。レポートの別のページを調べようとする場合、「 Next」又は「Previous」ボタンをクリックしてください。このダイアログボックスの中で、下記の操作を行うことができます。

  • Select page printing options（ページ印刷オプションの選択）：「Page Setup」ボタンをクリックします。

  • Print the standard report（標準レポートの印刷）：「View」のプルダウンリストから「Standard report（標準レポート）」（又は「Full report（フルレポート）」）を選択して、「Print」ボタンをクリックします。この時に「Print ダイアログボックス」が表示されて，「OK」ボタンをクリックすると、レポートを印刷することができます。ここでプリンターを変更した場合、プログラムは再度ページ印刷オプションを選択してくださいと提示しますので、ご注意ください。これは選択されたプリンターがすべてのページ印刷オプションをサポートすることを確保するためです。標準レポートは 3～4 ページで構成されて、表示内容が会社情報、コメント（ある場合）、メイングラフ及び主統計データを含んでいます。



    Print ダイアログボックス


  • Print the detailed report（詳しいレポートの印刷）：「View」のプルダウンリストから「Detailed report（詳しいレポート）」を選択して、「Print」ボタンをクリックします。この時に上記と同じな「Print ダイアログボックス」が表示されます。現在開けられた記録ファイルの生成時間が１日間を上回った場合、このオプションを選択することができます。詳しいレポートは標準レポートのすべてのページを含んで、また毎日のグラフと統計データを含んでいます。

  • Print the main graph only（メイングラフだけを印刷）：「View」のプルダウンリストから「Graph only（グラフだけ）」を選択して、「Print」ボタンをクリックします。この時に上記と同じな「Print ダイアログボックス」が表示されますが，印刷範囲を選択することができません。このオプションを利用して、グラフエリアに現在表示されているグラフを印刷することができます。

注：

  • 選択されたレポートの印刷用プリンターに入れられた紙が「Page Setupダイアログボックス」に指定されたものと一致しない場合、レポートを正しく印刷することができない恐れがあります。

  • 初めて新しいバージョンのプログラムを運行する時に、ページ印刷オプションのファイルを見つけることができない場合、或はコンピュータのデフォルトプリンターがすでに変更された場合、「Print Preview ダイアログボックス」を表示する前に、プログラムは「Page Setup ダイアログボックス」でページ印刷オプションを選択してくださいと提示します。これはレポートの表示と印刷が正しいことを確保するためです。

  • 「Print ダイアログボックス」にある「Properties」ボタンをクリックして、プリンターの印刷オプション、例えば印刷品質を変更することができます。しかしここでページ印刷オプション（紙の大きさと方向）を変更してはいけません。ページ印刷オプションは「Page Setup ダイアログボックス」からしか入手できませんが、別のところで変更しても、印刷出力にいかなる作用がありません。

  •  グラフが白紙に印刷されるので、「Select Channel ダイアログボックス」で白色を曲線の色とすることを避けてください。

  • もっと良いグラフ印刷効果を得るために、カラープリンターを使ってください。
