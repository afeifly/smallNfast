レポートの保存とロード

一つ又は多数の記録ファイルを開けた後に、これらのファイルに対して異なる設定を行わなければならない可能性があります。例えばレポートのタイプや時間周期の変更、異なったチャンネルの選択、リークポイントの定義及びグラフのコメントの追加。別の記録ファイルを開ける時、或はプログラムの再起動の時に、これらの設定がなくなります。これらの設定を保存するために、「Save Report（レポートの保存）」機能を使うことができます。保存した設定を調べようとする場合、「Load Report（レポートのロード）」機能を使うことができます。

レポートの保存

レポートを保存しようとする場合、メニューの「File -> Save Report（ファイル -> レポートの保存）」を選択してください。この時に「Save Report ダイアログボックス」が表示されて，その中で保存を選択することができます。

  • All settings only（すべての設定）

    現在開けられた記録ファイルに関連する情報だけを .rep ファイルの中に保存します。このようにすると、生成したファイルが最も小さいです。しかし、このレポートをロードする前に、先にこれらの記録ファイルを含むデータベースを開けなければなりません。

  • All settings plus the currently opened record file(s)（すべての設定と現在開けられた記録ファイル）

    すべての設定と現在開けられた記録ファイルが .rep ファイルの中に保存されます。このようにすると、生成したファイルが最も大きいです。しかし、このレポートをロードする前に、先にこれらの記録ファイルを含むデータベースを開ける必要がありません。この方式で保存されたレポートを別のコンピューターに転送して、それにロードすることもできます。



    Save report ダイアログボックス


レポートを保存しようとする場合、必要のオプションの側のラジオボタンをクリックして、「Save」ボタンをクリックしてください。この時に次の通りの別のダイアログボックスを表示して、その中でファイルを保存することができます。一つのわりに大きいレポートを保存するのに何分間を必要とする可能性があるので、ご注意ください。



Save report dialog


レポートのロード

レポートをロードしようとする場合、メニューの「File -> Load Report（ファイル -> レポートのロード）」を選択してください。この時に「Load Report ダイアログボックス」が表示されます。



Load report ダイアログボックス


わりに大きいファイルをロードするのに、何分間を必要とする可能性があります。ロードが完成した時に、プログラムは提示します。データベースを開けたまま、一つのレポートを保存して、また別のデータベースを開けた後にこのレポートをロードすると、このレポートに含められた記録ファイルが第二のデータベースに追加されます。
