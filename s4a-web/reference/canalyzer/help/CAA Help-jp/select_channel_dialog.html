Select channel ダイアログボックス


ツールバーにあるチャンネル選択ボタン（ ）をクリックしたり、メニューの「Tool -> Select Channel（ツール -> チャンネルの選択）」を選択したりすると、このダイアログボックスを開けることができます。このダイアログボックスを利用して、現在開けられた記録ファイルに含まれたチャンネルを調べて配置することができます。また、ここで Y 軸の設定を変更することができます。 

  • ダイアログボックスのトップで使用可能と選定したチャンネルを表示します。選定されたチャンネルだけグラフで表示されて、これらのチャンネルの測量値がグラフで曲線で表示されます。一本の選定チャンネルを追加しようとする場合、使用可能のチャンネルを選定して、「Select」ボタンをクリックしてください。一本の選定チャンネルをキャンセルしようとする場合、使用可能のチャンネルを選定して、「 Remove」ボタンをクリックしてください。選定チャンネルは多とも 4 つの異なった単位がありますが、グラフは多とも 4 本の縦軸（Y）があるためです。ご注意ください。多とも 6 本のチャンネルがありますが、これらのチャンネルの異なる単位の数量が4つを上回ってはいけません。

  • 一本の選定チャンネルを選択して、曲線の作成に用いられた色とスタイルを調べることができます。「Change Color」と「Change Style」ボタンをクリックすると、それぞれ色とスタイルを変更することができます。「Choose Line Color ダイアログボックス」からいかなる必要な色を選択することができます。選択可能のスタイルは二つがありますが、即ち、実線と点線です。

  • Y 軸の設定はダイアログボックスの底部に表示されています。あるY軸の設定を調べたり、変更したりしようとする場合、まずその番号の側のラジオボタンをクリックする必要があります。各 Y 軸について、下記の操作を行うことができます。

    • 「Unit」のプルダウンリストを使って目盛りの単位を変更します。

    • ラベルが「Automatic Scaling」のオプションをチェックして、自動目盛り機能を起動します。この機能を停止した後に、下記の操作を行うことができます。

    • 「From」欄に開始値を入力して、「To」欄に終了値を入力して、目盛りを設定します。終了値が開始値より大きいです。

    • 「Number of Steps」欄に正整数を入力して目盛りのステップ数を指定します。

ダイアログボックスの底部の三つのボタンの機能は次の通りです。

  • 「OK」ボタンをクリックすると、配置を確定してダイアログボックスを終了することができます。

  • 「Apply」ボタンをクリックすると、配置を確定してダイアログボックスを開けたままにすることができます。

  • 「Cancel」ボタンをクリックすると、配置をキャンセルしてダイアログボックスを終了することができます。
