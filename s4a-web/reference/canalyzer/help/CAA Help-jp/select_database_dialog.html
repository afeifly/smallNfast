Select Database ダイアログボックス



Select database ダイアログボックス


メニューの「File -> Select Database（ファイル -> データベースの選択）」を選択して、このダイアログボックスを開けることができます。このダイアログボックスはすべての使用可能のデータベースのリストを表示して、これらのデータベースに対して管理を行うことができます。これらのデータベースは異なったバージョン又はコピーの CSSoftや CS-Busで生成されることができます。一部のデータベースはプログラムが初めてスタートする時にすでにリストに追加された可能性があります。このダイアログボックスの中で、下記の操作を行うことができます。

  • Select a database for opening（一つのデータベースを選択して開ける）：リストから一つのデータベースを選択して、「Select」ボタンをクリックします。プログラムは開けの操作が成功したのか、提示します。一つのデータベースを開けると、「Select Record File ダイアログボックス」でその中に記憶されたすべての記録ファイルを調べることができます。下記の状況で、開けの操作が失敗します。 

    • データベースのファイルのパスが無効です。例えば、パスの中のファイルが有効なデータベースファイルではない、或は削除されたのです。この状況で、データベースのファイルのパスを確認するべきです。

    • データベースは別のプログラムに使われています（CSSoft や CS-Bus）。この状況で、このデータベースを使うプログラムを閉じて、再度やってみます。

  • Add a new database to the list（一つの新しいデータベースをリストに追加する）：「New」ボタンをクリックします。「Add Database Connection ダイアログボックス」が表示されます。この時にデータベースの名称と説明を入力することができます。



    Add database connection ダイアログボックス


    データベースファイルのパスを指定する必要がある場合、「Select」ボタンをクリックしてください。この時に「Open ダイアログボックス」が表示されます。CSSoft や CS-Bus インストールディレクトリ（例えば「C:\Program Files\cssoft」）の中のファイル「csmain.data」を選択します。



    Open ダイアログボックス


    「Add Database Connection ダイアログボックス」に戻る時に、「OK」ボタンをクリックするだけでデータベースをリストに追加することができます。

  • Edit a database（データベースの編集）：リストから一つのデータベースを選択して、「Edit」ボタンをクリックします。この時に「Edit Database Connection ダイアログボックス」が表示されて、その中にデータベースの名称、説明とファイルのパスを編集することができます。

  • Remove a database（データベースの削除）：リストから一つのデータベースを選択して、「Remove」ボタンをクリックします。選定したデータベースが開けられたものではない場合、すぐ削除されます。開けられた場合、プログラムは確認してくださいと提示して、「Yes」だけ答えた後にデータベースを削除します。
