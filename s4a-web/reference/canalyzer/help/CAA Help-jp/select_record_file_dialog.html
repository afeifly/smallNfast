Select record file ダイアログボックス


ツールバーにあるファイル選択ボタン（ ）をクリックしたり、メニューの「Tool -> Select File（ツール -> ファイルの選択）」を選択したりすると、このダイアログボックスを開けることができます。このダイアログボックスを利用して、現在開けられたデータベースに記憶されたすべての記録ファイルを調べることができます。開けられたデータベースがない場合、ここでいかなる記録ファイルを表示しません。一つのデータベースを開ける必要がある場合、「Select Database ダイアログボックス」に入ってください。

このダイアログボックスは下記の機能を提供しています。

  • Select a record file for viewing（一つの記録ファイルを選択してビューする）：リストから一つの記録ファイルを選択して、「Select」ボタンをクリックします。記録ファイルに記憶された測量値はグラフで表示されます。

  • Select two or more record files for viewing simultaneously（同時に二つ又は多数の記録ファイルを調べることを選択する）：リストから二つ又は多数の記録ファイルを選択して、「Combine」ボタンをクリックします。多数の記録ファイルが必要な場合、クリックしながら Ctrl ボタンを押してください。連続的な多数の記録ファイルを選択する必要がある場合、初めてファイルをクリックして、また最後のファイルをクリックしながら、 Shiftボタンを押してください。この機能は異なった記録ファイルに含まれている測量値に便利な比較方式を提供しました。

  • View record file information（記録ファイルの情報のビュー）：リストから一つの記録ファイルを選択して、「File Information」ボタンをクリックします。この時に記録ファイルの情報を表示するダイアログボックスが出てきます。



    Record file information ダイアログボックス


  • Export record files to a CS Data file（一つ又は多数の記録ファイルを一つの CSデータファイルへエックスポートする）：リストから一つの記録ファイルを選択して、「Export」ボタンをクリックします。わりに大きい記録ファイルをエックスポートするのに、何分間を必要とする可能性があります。プログラムはロードが完成したと提示します。



    export ダイアログボックス


  • Delete record files from database（データベースから記録ファイルを削除）：リストから一つ又は多数の記録ファイルを選択して、「Delete」ボタンをクリックします。プログラムは削除を確認してくださいと提示します。 削除操作が取り消すことができませんが、つまり、削除した後に記録ファイルを回復することができないので、ご注意ください。
