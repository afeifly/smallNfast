起動と終了

CAAをインストールした後に、デスクトップにあるCAA アイコン（ ）をダブルクリックしたり、スタートメニューで相応のアイコンを選択したりして、プログラムを起動します。 

  • プログラムは自動的にコンピュータにある CSSoft や CS-Bus データベースのデフォルト位置（例えば、「C:\Program Files\cssoft」）を検索して、下記の状況でそれを配置します。

    • 初めてプログラムを起動すること、或は、

    • 前回プログラムを閉じる時に、いかなるデータベースを配置していないこと。

    検索が完成した後に、プログラムは見つけた全ての有効なデータベース数を提示します。また、少なくとも一つのデータベースを見つけても、プログラムは第一のデータベースを接続してみます。「 OK」ボタンを押した後に、 CAA のメインウィンドウが見えるようになります。



    初めての起動情報


  • その他の状況で、プログラムは CSSoft や CS-Bus データベースを検索しませんが、この時に直接にメインウィンドウが見えます。しかし、プログラムは前回に閉じる時にアクセスしたデータベースを接続してみます。接続が失敗した場合、プログラムはデータベースの設定をチェックしてくださいと提示します。



    CAA メインウィンドウ


メニューの「File -> Exit（ファイル -> 終了）」を選択したり、ウィンドウの標準閉じボタンをクリックしたりすると、プログラムを終了することができます。この二種類の状況で、プログラムは閉じる前にユーザーの設定を保存します。
