ツールバー

ツールバーはメインウィンドウの右側にあります。ツールバーのボタンは下記の常用機能をアクセスするショートカットを提供しました。

  •   － 現在開けられたデータベースに記憶されたすべての記録ファイルを表示して、一つ又は多数のファイルを選択して、グラフで調べることができます。もっと多い情報は、「Select Record File ダイアログボックス」を参照してください。

     

  •  － タイトル、軸名と凡例を含め、グラフに表示されるテキストをカスタマイズします。

     

  •  － 統計レポートを調べて、レポートには計算設定と測量データによって算出されたシステムの静態値を表示します。

     

  •  － グラフに表示された測量値の周期を指定します。もっと多い情報は、「時間周期の選択」を参照してください。

     

  •  － グラフに表示されたチャンネルを選択します。もっと多い情報は、「Select Channel ダイアログボックス」を参照してください。

     

  •   － エネルギーと圧縮空気のコストの計算に用いられる設定を指定します。もっと多い情報は、「Calculation Setting ダイアログボックス」を参照してください。

     

  •  － グラフで一本のリーク線を表示して、システムの平均的なリーク量を定義するのに用いられます。当該メニューを再選択すると、当該リーク線を隠します。

     

  •   － 会社の基本的な情報を入力します。もっと多い情報は、「基本的な情報の入力」を参照してください。

     

  •  － グラフにコメントを追加します。もっと多い情報は、「グラフコメントの追加」を参照してください。

     

  •   －レポートのプレビューと印刷。もっと多い情報は、「レポートのプレビューと印刷」を参照してください。

     

  •   － 同じなグラフで異なった日数と週間数の測量値を調べます詳しくは「同じなグラフで異なった日数と週間数の測量値を調べる」を参照してください。

     

  •   －一日間、一週間又は一ヶ月間のグラフのビューを選択します。また、1分間、15分間又は1時間あたりの各チャンネルの平均的な測量値を選択することができて、表示される値の数量を減らします。
