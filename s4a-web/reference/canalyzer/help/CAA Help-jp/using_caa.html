CAAの使用

CAA は Microsoft Windows 環境で運行しているグラフアプリケーションです。現在このプログラムは英語のユーザーインターフェースだけをサポートしています。プログラムの後続バージョンはその他の言語のユーザーインターフェースをサポートします。

スクリーンショットに関して

重要な可視のコンポーネントを備えるプログラムを説明することと同じで、当ファイルはたくさんの CAA のスクリーンショットを提供しました。これらのスクリーンショットは異なったバージョンのプログラムでから生まれたものです。このため、それらは現在のバージョンのスクリーンショットとある程度に異なっている可能性があります。しかし、これらの違いはとても小さいので、読者に決して困らせません。
