空压系统分析软件 (CAA)

使用记录的测量值分析压缩空气系统的应用程序

压缩空气分析器（简称 CAA）是一个使用储存在 CSSoft 或 CS-Bus 数据库中的测量值来分析压缩空气系统的应用程序。通过这一程序，可以图表或统计的形式查看测量值，从而对压缩空气系统有一个全面的了解。

CAA 提供了下列特性：

 

•         管理由不同版本或副本的 CSSoft 或 CS-Bus 产生的多个数据库

•         一次连接一个数据库并从中提取所有记录文件

•         以图表形式显示储存在单个或多个记录文件中的测量值

•         定制图表视图

•         定制泄漏设置以及公司信息

•         根据用户定义的设置生成有关测量值的统计数据

•         打印一份包含公司信息、注释、图表以及统计数据的报表

•         将报表全部或部分导出到 JPEG 或 PDF 文件中

•         将记录文件导入到当前打开的数据库中

•         将数据库中的记录文件导出到一个 CS 数据文件中

•         将报表保存为一个 .rep 文件，并在将来加载

•         在同一个图表内查看不同天数和周数的测量值
