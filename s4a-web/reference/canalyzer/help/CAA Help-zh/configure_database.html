配置数据库和查看记录文件

启动程序后，可能需要对 CAA 访问的数据库进行配置。为此，请选择菜单项“File -> Select Database（文件 -> 选择数据库）”以打开“Select Database对话框”。通过这一对话框，可查看可用的数据库，选择需要打开的数据库，或对数据库进行添加、编辑或删除操作。

打开一个数据库后，可选择储存在其中的一个或多个记录文件，以便在图表区中查看。为此，请单击工具栏上的选择文件按钮( )打开“Select File 对话框”。通过这一对话框，可查看储存在该数据库中的所有记录文件，并选择一个或多个文件进行查看。有关图表区特性的详细信息，请参见“图表区”。
