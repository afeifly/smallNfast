导入记录文件

可将包含在一个 CS 数据文件（.csd）中的记录文件导入到当前打开的数据库中。CS 数据文件由 CSSoft 或 CS-Bus 生成，包含一个或多个记录文件。通过这一功能，可查看其它计算机获得的测量值。

如需进行导入操作，请选择菜单项“File -> Import Data（文件 -> 导入数据）”。此时将出现“Import 对话框”。



Import 对话框


导入一个较大的文件可能需要几分钟的时间。程序会显示导入结果。导入成功后，所有包含在该 CS 数据文件中的记录文件均会被导入，并显示在“Select Record File 对话框”中。
