主窗口概述

下图显示了一个典型的 CAA 主窗口。



一个典型的 CAA 主窗口


 

    • 菜单栏包含五个菜单，可进行不同的操作。有关这些菜单的详细描述，请参见“菜单”。
    • 工具栏的大多数功能与菜单功能相同，但这些按钮提供了一种访问程序基本特性的快捷方式。如需查看某个按钮的功能，请将鼠标置于其上，停留一会儿后将显示简要的帮助信息解释这一按钮的功能。有关这些工具栏按钮的详细描述，请参见“工具栏”菜单。
    • 图表区显示当前打开的记录文件的图表。图表区具有如下特性：
      • 图表具有一条横轴（时间轴）和最多 4 条纵轴。刻度单位显示在每条纵轴的旁边。

      • 图表上的每条曲线代表某个特定通道的测量值，其颜色与通道名称的颜色相同。图表最多可显示 6 条通道的测量值，但这些通道的不同单位的数量不得超过 4 个。可在“Select Channel 对话框”中单击工具栏上的选择通道按钮（ ）更改图表中的显示通道。

      • 可单击工具栏上的更改文字按钮（ ）或选择菜单项“Setting -> Change Text（设置 -> 更改文字）”定制图表上显示的文字，包括标题、轴名称和图例（即通道名称和单位）。除图例外，所有输入的文字都会被保存，并在下次程序启动时加载。

 


Change Text对话框


      • 如记录文件的周期太长，可单击右下角的“--->”或“<---”按钮查看图表的不同部分。
      • 可单击工具栏上的选择报表类型按钮（ ）或选择菜单项“View -> Select Report Type（视图 -> 选择报表类型）”打开“Select Report Type 对话框”，然后选择查看一天、一周或一个月的图表。

默认情况下，每条可见通道的所有测量值均显示在图表区。如在一个较短的周期内显示了过多的值，曲线可能会变得模糊。如需更清晰地查看曲线，可在该对话框中选择计算每1分钟、15分钟或1小时各通道的平均测量值，从而减少所显示的值的数量。



Select report type 对话框



    • 可单击工具栏上的选择时间周期按钮（ ）或选择菜单项“View -> Select Time Period（视图 -> 选择时间周期）”指定显示在图表上的测量值的周期。Time（时间）轴上的刻度根据指定的周期而变化。如需还原图表，请选择菜单项“View -> Reload（查看 -> 重新加载）”。



      Time option 对话框


    • 可单击工具栏上的添加注释按钮（ ）或选择菜单项“Tool -> Add Comment To Graph（工具 -> 添加图表注释）”，在图表上添加注释。添加了注释后，可将其拖放在图表的任意位置。此外，也可编辑和删除注释。右击注释并选择相应的菜单项即可。请注意，所添加的注释在打印或导出图表时也会出现。

    • 可右击图表区然后选择“Reload”，或选择菜单项“View -> Reload（视图 -> 重新加载）指定显示在图表上的测量值的周期。
