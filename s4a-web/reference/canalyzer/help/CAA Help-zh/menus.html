菜单

File 菜单

  • Select File（选择文件）－显示储存在当前打开的数据库中的所有记录文件，可选择一个或多个文件以图表形式进行查看。有关更多信息，请参见“Select Record File 对话框”。
  • Select Database（选择数据库）－显示所有可用的数据库，可选择一个打开。有关更多信息，请参见“Select Database 对话框”Select record file 对话框。
  • Save Report（保存报表）－将与当前打开的记录文件相关联的信息保存到一个 .rep 文件中。有关更多信息，请参见“保存和加载报表Select Database 对话框”。
  • Load Report（加载报表）－从一个 .rep 文件加载与记录文件相关联的信息。有关更多信息，请参见保存和加载报表“保存和加载报表”。
  • Import Data（导出数据）－将一个 CS 数据文件（.csd）导入到当前打开的数据库中。有关更多信息，请参见“导入记录文件”Select record file 对话框。
  • Export To PDF（导出到PDF文件）－将生成的报表全部或部分导出到 JPEG 或 PDF 文件中。有关更多信息，请参见“将报表导出到 JPEG 或 PDF 文件中”。
  • Page Setup（页面设置）－选择报表的页面打印选项。有关更多信息，请参见“选择页面打印选项”Select record file 对话框。
  • Print Preview（打印预览）－预览和打印报表。有关更多信息，请参见预览和打印报表选择页面打印选项“预览和打印报表”。
  • Print（打印）－预览和打印报表。有关更多信息，请参见预览和打印报表选择页面打印选项“预览和打印报表”。
  • Exit（退出）－退出应用程序。退出时自动保存用户设置。

Setting 菜单

  • Change Text（更改文字）－定制图表上显示的文字，包括标题、轴名称和图例。

  • Calculation Setting（计算设置）－指定用于计算能源和压缩气体成本的设置。有关更多详情，请参见“Calculation Setting 对话框”。

  • Basic Setting（基本设置）－输入公司的基本信息。有关更多信息，请参见“输入基本信息预览和打印报表”。

Tool 菜单

  • Show Statistics（显示统计数据）－查看统计报表，报表中显示根据计算设置和测量数据计算出的系统静态值。
  • Define Leakage（定义泄漏）－在图表上显示一条泄漏线用以定义系统的平均泄漏量。Selecting this item again will hide the leakage line.
  • 再次选择这一菜单项将隐藏该泄漏线。
  • Select Channel（选择通道）－选择显示在图表上的通道。有关详情，请参见“Select Channel 对话框”。
  • Compare Data（比较数据）－在同一个图表内查看不同天数和周数的测量值。有关详情，请参见“在同一个图表内查看不同天数和周数的测量值”。
  • Add Comment To Graph（添加图表注释）－在图表中添加注释。有关更多信息，请参见“添加图表注释预览和打印报表”。
  • Write Comment Page（编写注释页）－在注释页中输入注释，这些注释将被包含在打印/导出的报表中。

View 菜单

  • Select Report Type（选择报表类型）－选择查看一天、一周或一个月的图表。此外，还可选择计算每1分钟、15分钟或1小时各通道的平均测量值，从而减少所显示的值的数量。

  • Select Time Period（选择时间周期）－指定显示在图表上的测量值的周期。有关更多信息，请参见“选择时间周期”。

  • Reload（重新加载）－重新加载当前记录文件，并将图表还原至其初始状态。

  • Display Records At Cursor（显示光标位置记录）－对光标位置所在的时间和可用测量值启用动态显示功能。时间和测量值显示在图表区的顶部。再次选择这一菜单项将停用该功能。

Help 菜单

  • Help Contents（帮助内容）－显示应用程序的帮助内容。

  • Register Software（注册软件）－查看许可信息和注册软件。

  • About（关系）－显示版本信息（如需用户支持，必须提供版本信息）。
