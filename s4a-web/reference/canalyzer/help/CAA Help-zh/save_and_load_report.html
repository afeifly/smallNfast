保存和加载报表

打开一个或多个记录文件后，可能需要对这些文件进行不同的设置，如更改报表类型或时间周期，选择不同的通道，定义泄漏点以及添加图表注释。在打开另一个记录文件或重新启动程序时，这些设置将会丢失。为了保留这些设置，可使用“Save Report（保存报表）”功能。如需查看保存的设置，可使用“Load Report（加载报表）”功能。

保存报表

如需保存报表，请选择菜单项“File -> Save Report（文件 -> 保存报表）”。此时将出现“Save Report 对话框”，其中可选择保存：

  • All settings only（所有设置）

    只有与当前打开的记录文件相关联的信息会被保存到一个 .rep 文件中。这样生成的文件最小。但是，在加载这一报表前，必须先打开包含这些记录文件的数据库。

  • All settings plus the currently opened record file(s)（所有设置以及当前打开的记录文件）

    所有设置以及当前打开的记录文件都会被保存到一个 .rep 文件中。这样生成的文件最大。但是，在加载这一报表前，无需先打开包含这些记录文件的数据库。以这种方式保存的报表也可以传送到另一台计算机并加载到其中。



    Save report 对话框


如需保存报表，请点击所需选项旁边的单选按钮，然后单击“Save”按钮。此时可以看到如下所示的另一个对话框，其中可对文件进行保存操作。请注意，保存一个较大的报表可能需要几分钟的时间。



Save report 对话框


加载报表

如需加载报表，请选择菜单项“File -> Load Report（文件 -> 加载报表）”。此时将出现“Load Report 对话框”。



Load report 对话框


加载一个较大的报表可能需要几分钟的时间。程序会提示加载完成。如在数据库打开时保存了一个报表，并在打开了另一个数据库后加载这一报表，则包含在该报表中的记录文件将被添加到第二个数据库。
