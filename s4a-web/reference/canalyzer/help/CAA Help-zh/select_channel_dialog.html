Select Channel 对话框



Select channel 对话框


单击工具栏上的选择通道按钮（ ）或选择菜单项“Tool -> Select Channel（工具 -> 选择通道）”，可打开这一对话框。通过该对话框，可查看和配置包含在当前打开的记录文件中的通道。此外，还可在此处更改 Y 轴的设置。

  • 对话框的顶部显示可用和选定的通道。只有选定的通道会显示在图表上，这些通道的测量值会在图表上以曲线的形式显示。如需添加一条选定通道，请选择一条可用通道，然后单击“Select”按钮。如需移除一条选定通道，请选择一条选定通道，然后单击 “Remove”按钮。请注意，选定通道最多只能有 4 个不同的单位，因为图表最多只能有 4 条纵轴（Y）。最多可以有 6 条通道，但这些通道的不同单位的数量不得超过 4 个。

  • 可选择一条选定通道，以查看描绘曲线所使用的颜色和样式。单击“Change Color”和“Change Style”按钮可分别更改颜色和样式。可从“Choose Line Color 对话框”选择任何所需的颜色。有两种样式可供选择：实线和虚线。

  • Y 轴的设置显示在对话框的底部。如需查看或更改某条 Y 轴的设置，首先需要点击其编号旁边的单选按钮。对于每条 Y 轴，可进行下列操作：

    • 使用“Unit”下拉列表更改刻度单位。

    • 勾选标签为“Automatic Scaling”的选项以启用自动刻度功能。停用这一功能后，可进行下列操作：

    • 通过在“From”栏输入起始值以及在“To”栏输入结束值来设置刻度。结束值应大于起始值。

    • 通过在“Number of Steps”栏中输入一个正整数来指定刻度的步数。

对话框底部三个按钮的功能如下：

  • 单击“OK”按钮可应用配置并关闭对话框。

  • 单击“Apply”按钮可应用配置并保持对话框打开。

  • 单击“Cancel”按钮可取消配置并关闭对话框。
