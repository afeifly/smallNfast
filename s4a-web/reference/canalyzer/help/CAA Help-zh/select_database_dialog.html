Select Database 对话框



Select database 对话框


选择菜单项“File -> Select Database（文件 -> 选择数据库）”可打开这一对话框。该对话框显示所有可用数据库的列表，可对这些数据库进行管理。这些数据库可由不同版本或副本的 CSSoft 或 CS-Bus 产生。某些数据库可能在程序首次启动时就已添加到列表中。在这一对话框中，可进行下列操作：

  • Select a database for opening（选择打开一个数据库）：在列表中选择一个数据库，然后单击“Select”按钮。程序会提示打开操作是否成功。一旦打开了一个数数据库，就可在“Select Record File 对话框”中查看储存在其中的所有记录文件。下列情况下，打开操作将会失败：

    • 数据库的文件路径无效，例如，路径中的文件不是有效的数据库文件或已被删除。在这种情况下，应确认数据库的文件路径。

    • 数据库正在被另一个程序使用（CSSoft 或 CS-Bus）。在这种情况下，应关闭使用这一数据库的程序，然后重试。

  • Add a new database to the list（添加一个新的数据库到列表）：单击“New”按钮。将出现“Add Database Connection 对话框”。此时可输入数据库的名称和描述。



    Add database connection 对话框


    如需指定数据库文件路径，请单击“Select”按钮。此时将出现“Open 对话框”。选择 CSSoft 或CS-Bus 安装目录（如“C:\Program Files\cssoft”）中的文件“csmain.data”。



    Open 对话框


    返回到“Add Database Connection 对话框”时，只需单击“OK”按钮即可将数据库添加到列表中。

  • Edit a database（编辑数据库）：在列表中选择一个数据库，然后单击“Edit”按钮。此时将出现“Edit Database Connection 对话框”，其中可编辑数据库的名称、描述和文件路径。

  • Remove a database（移除数据库）：在列表中选择一个数据库，然后单击“Remove”按钮。如选定的数据库不是打开的，则会即时被移除。如果是打开的，则程序会提示进行确认，并只有在回答“Yes”后才将数据库移除。
