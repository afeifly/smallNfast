Select Record File 对话框



Select record file 对话框


单击工具栏上的选择文件按钮（ ）或选择菜单项“File -> Select File（文件 -> 选择文件）”，可打开这一对话框。通过这一对话框，可查看储存在当前打开的数据库中的所有记录文件。如没有数据库是打开的，则此处不会显示任何记录文件。如需打开一个数据库，请转至“Select Database 对话框”。

此对话框提供下列功能：

  • Select a record file for viewing（选择查看一个记录文件）：在列表中选择一个记录文件，然后单击“Select”按钮。储存在记录文件中的测量值会以图表的形式显示。

  • Select two or more record files for viewing simultaneously（选择同时查看两个或多个记录文件）：在列表中选择两个或多个记录文件，然后单击“Combine”按钮。如需选择多个记录文件，请在单击的同时按住 Ctrl键。如需选择连续的多个记录文件，请单击第一个文件，然后在单击最后一个文件的同时按住 Shift键。该功能为包含在不同记录文件中的测量值提供了一种方便的比较方式。

  • View record file information（查看记录文件信息）：在列表中选择一个记录文件，然后单击“File Information”按钮。此时将弹出一个显示记录文件信息的对话框。



    Record file information 对话框


  • Export record files to a CS Data file（导出一个或多个记录文件到一个 CS 数据文件）：在列表中选择一个记录文件，然后单击“Export”按钮。导出较大的记录文件可能需要几分钟的时间。程序会提示导出完成。



    export dialog-c.jpg


  • Delete record files from database（从数据库删除记录文件）：在列表中选择一个或多个记录文件，然后单击“Delete”按钮。程序会提示对删除进行确认。请注意，删除操作是无法撤销的，也就是说，删除后将无法恢复记录文件。
