启动和结束

成功安装 CAA 后，可双击桌面上的 CAA 图标（ ），或在开始菜单选择相应的图标，以启动程序。

  • 程序会自动搜索计算机中 CSSoft 或 CS-Bus 数据库的默认位置（如“C:\Program Files\cssoft”），并在下列情况下对其进行配置：

    • 首次启动程序，或t

    • 上次关闭程序时未配置任何数据库

    搜索完成后，程序将提示所找到的有效数据库的数量。此外，如找到至少一个数据库，程序也会尝试连接至第一个数据库。按下“OK”按钮后，将看到 CAA 的主窗口。



    首次启动信息


  • 在其它情况下，程序不会搜索 CSSoft 或 CS-Bus 数据库，此时将直接看到主窗口。但是，程序会尝试连接至上次关闭时正在访问的数据库。如尝试失败，程序将会提示检查数据库的设置。



    CAA 主窗口


选择菜单项“File -> Exit（文件 -> 退出）”或单击标准关闭窗口按钮，可退出程序。在这两种情况下，程序均会在关闭前保存用户设置。
