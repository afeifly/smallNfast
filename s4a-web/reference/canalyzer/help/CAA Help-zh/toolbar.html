工具栏

工具栏位于主窗口的右侧。工具栏按钮提供了一种访问下列常用功能的快捷方式。

  •   － 显示储存在当前打开的数据库中的所有记录文件，可选择一个或多个文件以图表形式进行查看。有关更多信息，请参见“Select Record File 对话框”Select record file 对话框。

     

  •  － 定制图表上显示的文字，包括标题、轴名称和图例。

     

  •  － 查看统计报表，报表中显示根据计算设置和测量数据计算出的系统静态值。

     

  •  － 指定显示在图表上的测量值的周期。有关更多信息，请参见选择时间周期“选择时间周期”。

     

  •  － 选择显示在图表上的通道。有关详情，请参见“Select Channel 对话框”Select Channel 对话框。

     

  •   － 指定用于计算能源和压缩气体成本的设置。有关更多详情，请参见“Calculation Setting 对话框”Calculation Setting 对话框。

     

  •  － 在图表上显示一条泄漏线用以定义系统的平均泄漏量。再次选择这一菜单项将隐藏该泄漏线。

     

  •  － 输入公司的基本信息。有关更多信息，请参见输入基本信息“输入基本信息”。

     

  •  － 在图表中添加注释。有关更多信息，请参见“添加图表注释添加图表注释”。

     

  •  － 预览和打印报表。有关更多信息，请参见“预览和打印报表预览和打印报表”。

     

  •  － 在同一个图表内查看不同天数和周数的测量值。有关详情，请参见“在同一个图表内查看不同天数和周数的测量值”在同一个图表内查看不同天数和周数的测量值。

     

  •   － 选择查看一天、一周或一个月的图表。此外，还可选择计算每1分钟、15分钟或1小时各通道的平均测量值，从而减少所显示的值的数量。
