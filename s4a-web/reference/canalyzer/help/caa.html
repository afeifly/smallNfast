Compressed Air Analyzer (CAA)

An application for analyzing compressed air system using recorded measurement values

Compressed Air Analyzer, or CAA for short, is a program designed for analyzing compressed air system using measurement values stored in CSSoft or CS-Bus databases. The program allows you to examine the measurement values graphically and statistically, thus giving you a full picture of your compressed air system.

CAA offers the following features:

  • Managing multiple databases generated by different versions or copies of CSSoft or CS-Bus

  • Connecting to one database at a time and retrieving all record files from it

  • Presenting measurement values stored in a single or multiple record files in a graphic form

  • Customizing leakage settings and your company information

  • Generating statistics for measurement values based on user-defined settings

  • Printing a report presenting your company information, the graph and the statistics

  • Exporting the graph and/or the Statistics report to JPEG or PDF file(s)

  • Importing record files into the currently opened database

  • Exporting record files in the database to a CS Data file
