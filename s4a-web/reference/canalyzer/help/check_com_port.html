Checking COM port connected to the device

In order to let CSSoft communicate with a CS device such as Interface Box and CS 2390, you need to tell the program which COM port the device is connected to.

To check the COM port connected to the device, follow these steps:

  • Make sure the CS device is connected to your computer and the driver for the device has been installed.

  • Right-click the My Computer icon on your desktop or in the Start menu and select Properties to open the "System Properties" dialog.


    (Missing picture [right-click my computer.jpg])
    My Computer icon


  • On the "System Properties" dialog, select the Hardware tab and click "Device Manager" button.


    (Missing picture [System Properties.Hardware.jpg])
    System Properties dialog


  • On the "Device Manager" dialog, expand "Ports (COM & LPT)" section in the list to find the CS device ("CS instruments USB-SDI interface" in this example).


    (Missing picture [Device Manager.jpg])
    Device Manager dialog


  • The COM port (COM9 in this example) will be shown in the brackets.
