Compacting database

You can use the menu item "Tool -> Compact Database" to compact the program's database. This function helps reduce the database's size and access time after you have added or deleted a lot of files to/from the database.

Note:

  • If there is recording going on, you should stop it before selecting this menu item. Otherwise the program will stop it before starting the compacting process.

  • Compacting large database can take up to several minutes and cannot be interrupted.
