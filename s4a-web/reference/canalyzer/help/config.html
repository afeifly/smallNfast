Configuring the program

Through the menu item "Configuration -> Communication", you can open a dialog for changing the communication options.


(Missing picture [config dialog.jpg])
Dialog for changing the communication options


The following settings can be configured:

  • COM port number: The number of the COM port the CS device is connected to. For instructions to check the COM port, please refer to Checking COM port connected to the device.

  • Device: The type of CS device you are using with the program. Available choices are: CS2390, DS 300, FA300, VA300 and Flow Processor.

Note: If the device is DS 300, there is no need to select a COM port.
