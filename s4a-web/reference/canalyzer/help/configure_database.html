Configuring databases and viewing record files

After starting the program, you may need to configure the databases accessed by CAA. To do this, click the database toolbar button () to open Select database dialog. This dialog allows you to view the available databases, select one for opening, and add, edit and remove databases.

After opening a database, you can select one or more record files stored in it for viewing in the graph area. To do this, click the Select file toolbar button () to open Select record file dialog. This dialog allows you to view all records files stored in the database and select one or more for viewing. For detailed information about the features provided for the graph area, see Graph area.
