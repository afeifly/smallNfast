Compressed Air Analyzer

An application for

CSSoft is a program designed for acquiring, recording and displaying measurement values from CS sensors or devices. By running this program on an average computer, you can easily view the current and recorded measurement values. These measurement values can be stored as separate files in database and exported to Excel files.

CSSoft features in particular:

  • Acquiring current measurement values from CS sensors including FA 300 and VA 300, via another device such as Interface Box.

  • Retrieving and deleting recorded measurement values stored in CS devices with logging function including DS 300 and CS 2390

  • Recording current measurement values from CS sensors

  • Storing retrieved and recorded measurement values as separate files in database

  • Presenting retrieved and recorded measurement values in graphic and tabular forms

  • Exporting files in database to Microsoft Excel files or CS Data files

  • Importing CS Data files into database

  • Printing measurement values presented in graphic form

Note:

CSSoft can be used with only one CS device, which can be an Interface Box, a DS 300 or a CS 2390. You can use another program called CS-Bus Basic to communicate with multiple DS 300.
