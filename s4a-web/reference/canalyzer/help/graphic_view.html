Graphic view


(Missing picture [graphic view-c.jpg])
Graphic view


You can switch the program to graphic view by selecting the menu item "View -> Graphic View" or clicking the toolbar button ([null]).

The program shows a graph representing the retrieved or recorded measurement values in graphic view. The following features are provided in graphic view:

  • The graph has one horizontal axis (Time axis) and one or two vertical axes (Left Y axis and Right Y axis). The unit for the scale is shown above each vertical axis.

  • Each curve represents the measurement values for a particular channel, and is in the same color as the channel's name. The graph can show measurement values for up to 10 channels as long as the number of different units for these channels doesn't exceed two. You can change the channels visible on the graph in View configuration dialog for graphic view.

  • You can click the "Show Online Value" button to start recording and viewing current measurement values. You can perform this operation when the device is online or offline. If the device is offline, it will be set online. To stop the recording, click the toolbar button ([null]) or select the menu item "Tool -> Stop Recording". Refer to Recording current measurement values for details.

  • You can click the "Open File" button, the Open file toolbar button ([null]) or select the menu item "File -> Open File" to view all protocol headers stored in the program's database and select one for viewing. Refer to Protocol header dialog for more information.

  • You can click the View configuration toolbar button ([null]) or select the menu item "Configuration -> View" to view and change the view configuration for graphic view. See View configuration dialog for graphic view for details.

  • You can right-click anywhere on the graph and select "Add Comment" to add a comment to the graph. After the comment is added, you can drag it anywhere on the graph. A comment can be edited and removed. Just right-click it and select the corresponding menu item. Note that the added comments will also appear when you print the graph.

  • You can press and hold the left mouse button, draw a box around a portion of the graph and release the mouse button to zoom in on the graph. The program will zoom in on all the curves within the selected period of time. You can zoom in multiple times. To fully zoom out on the graph, i.e. restore its view, right-click on the graph and select "UnZoom".

  • You can click the "Select Time Period" button to change the period over which measurement values are shown on the graph. The scale on the Time axis will change according to the selected period. This function is similar to the zoom function. To restore the graph, right-click on the graph and select "UnZoom".
