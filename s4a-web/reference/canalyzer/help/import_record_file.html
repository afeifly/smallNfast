Importing record files

You can import record files contained in a CS Data (.csd) file into the currently opened database. A CS Data file can be generated using CSSoft or CS-Bus to contain one or more record files. Using this function, you can examine measurement values acquired by other computers.

To initiate the import, select the menu item "File -> Import Data". The Import dialog will appear.



Import dialog


Importing a large file may take up to several minutes. The program will notify you of the result of the import. If the import is successful, all record files contained in the CS Data file will be imported and will be shown at the end of the list in Select record file dialog.
