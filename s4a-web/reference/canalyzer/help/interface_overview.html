Main interface overview

CSSoft's main interface consists of menu bar, toolbar, main panel and status bar. The program provides three different views: Online view, Graphic view and Table view. To switch the view, use the three toolbar buttons ([null], [null] and [null]) or the View menu. Please note that graphic view and table view always show the same set of measurement values for the same channel, though in different forms.   • Menu bar contains six menus, which allows different operations. See Menus for detailed descriptions of these menus.

  • Tool bar replicates most menu functions, but the buttons provide quick access to the basic features of the program. To see what a button does, hold the mouse over it for a moment and brief help will appear, explaining that button's function.

  • Main panel displays a simulated panel meter in Online view, a graph in Graphic view and a table in Table view.

  • Status bar displays the status of the device the program is communicating with.
