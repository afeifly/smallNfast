Logger memory header list dialog


This dialog is for DS 300 only and can be opened by clicking the icon ([null]), clicking the toolbar button ([null]) or selecting the menu item "Device -> Read Recorded Data From Device". The dialog allows you to view the list of all protocol headers stored in DS 300 and provides the following functions:

  • View protocol header information: select a protocol header in the list and click "Header Information" button. A dialog will show up displaying the header information.


    (Missing picture [logger header info.jpg])
    Protocol header information dialog


  • Read a single protocol header: select a protocol header in the list and click "Read" button. A dialog will appear showing the reading progress. When the reading is complete, you will be prompted to enter a filename and save the file to the database or you may cancel saving the file. After the file is saved, the protocol header will be available as the last item in the list in Protocol header dialog. Please note that reading a large protocol header can take up to several minutes.

  • Read all protocol headers: simply click "Read All" button. This function works in the same way as the "Read" function, except that all protocol headers in the list will be read and you will be prompted to save the files one by one when the reading is complete.

  • Delete protocol headers: click to select one or more protocol headers in the list and click "Delete Selected Protocol Header" button. You will be asked to confirm the deletion. To Select multiple protocol headers, click while holding the Ctrl key. To select multiple protocol headers in a rows, click the first one and then click the last one while holding the Shift key. Please note that the deletion is final, i.e. after that it is not possible to bring the protocol header back.
