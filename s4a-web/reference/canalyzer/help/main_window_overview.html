Main window overview

The figure below shows what a typical CAA main window looks like.



A typical CAA main window


  • Menu bar contains five menus, which allows different operations. See Menus for detailed descriptions of these menus.

  • Toolbar replicates most menu functions, but the buttons provide quick access to the basic features of the program. To see what a button does, hold the mouse over it for a moment and brief help will appear, explaining that button's function. See Toolbar for detailed descriptions of the toolbar buttons.

  • Graph area displays the graph for the selected record file. The following features are provided for the graph area:

    • The graph has one horizontal axis (Time axis) and two vertical axes (Left Y axis and Right Y axis). The unit for the scale is shown beside each vertical axis.

    • Each curve on the graph represents the measurement values for a particular channel, and is in the same color as the channel's name. The graph can show measurement values for up to 6 channels as long as the number of different units for these channels doesn't exceed two. You can click the Select channel toolbar button () to change the channels visible on the graph in Select channel dialog.

    • You can click the Change text toolbar button () or select the menu item "Setting -> Change Text" to customize the text that appears on the graph including title, axis names and legends (i.e. channel names and units).



      Change text dialog


    • If record file's period is too long, you can click "Next ->" or "<- Previous" in bottom right corner to view different sections of the graph.

    • You can select the menu item "View -> Select Report Type" to open Select Report Type dialog and choose to view the graph over a period of one day, one week or one month.

      By default all measurement values of every visible channel are displayed in the graph area. When too many values are displayed over a relatively short peroid of time, the curve may become fuzzy. To view the curve more clearly, you can reduce the number of displayed values by choosing to average every channel' the measurement values every 1 minute, 15 minutes or 1 hour in this dialog.

    • You can click the Select time period toolbar button () or select the menu item "View -> Select Time Period" to specify the period over which measurement values are shown on the graph. The scale on the Time axis will change according to the specified period. To restore the graph, select the menu item "View -> Reload".



      Time option dialog


    • You can click the Add comment toolbar button () to add a comment to the graph. After the comment is added, you can drag it anywhere on the graph. A comment can be edited and removed. Just right-click it and select the corresponding menu item. Note that the added comments will also appear when you print or export the graph.

    • You can select the menu item "View -> Reload" to reload the current record file and restore the graph to its initial state.
