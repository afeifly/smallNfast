Menus

File menu

  • Select File - displays all record files stored in the currently opened database and lets you select one or more for viewing in a graphic form. Refer to Select record file dialog for more information.

  • Select Database - displays all available databases and lets you select one for opening. Refer to Select database dialog for more information.

  • Import Data - imports a CS Data (.csd) file into the currently opened database. Refer to Importing record files for more information.

  • Page Setup - selects page printing options for the report. For more information, see Selecting page printing options.

  • Print Preview - previews and prints the report. For more information, see Previewing and Printing report.

  • Print - previews and prints the report. For more information, see Previewing and Printing report.

  • Exit - exits this program. User settings are saved automatically when you exit.

Setting menu

  • Change Text - customizes the text that appears on the graph including title, axis names and legends.

  • Calculation Setting - specifies the settings for calculating the cost of leakage. For more details, see Calculation setting dialog.

  • Basic Setting - enters the basic information about your company. For more information, see Entering basic information.

Tool menu

  • Define Leakage - brings out a red leakage line on the graph to define the leakage point. Selecting this item again will hide the leakage line.

  • Select Channel - selects the channels that will appear on the graph. See Select channel dialog for details.

  • Add Comment - adds a comment to the graph. For more information, see Adding comments to graph.

View menu

  • Select Report Type - chooses to view the graph over a period of one day, one week or one month.

  • Select Time Period - specifies the period over which measurement values are shown on the graph. For more information, see Selecting time period.

  • Reload - reloads the current record file and restores the graph to its initial state.

Help menu

  • Help Contents - displays the help contents for this program.

  • About - shows the version information (important to know if you'll be requesting user support).
