Online view


You can switch the program to online view by selecting the menu item "View -> Online View" or clicking the toolbar button ([null]).

The program shows a panel meter representing the actual device in online view. To use most functions in this view, you need to set the device online by clicking the Go online toolbar button ([null]) or selecting "Device -> Go Online" menu item. Once the device is online, the program will periodically acquire current measurement values from the device and display them on the panel meter. The following features are provided in online view:

  • The panel meter displays two channels' measurement values at a time. If the device has more than two channels configured, you need to click the icon ([null]) or ([null]) to view other channels' measurement values.

  • You can click the icon ([null]) or select the menu item "Configuration -> Device" to view the device details and edit some of them if needed.

  • You can click the icon ([null]), the toolbar button ([null]) or select the menu item "Device -> Read Recorded Data From Device" to retrieve recorded measurement values stored in a CS device with logging function including DS 300 and CS 2390.

    • When the connected device is CS 2390, the program will start reading all measurement values stored in the device immediately.

    • When the connected device is DS 300, the program will display all protocol headers stored in the device and let you select one or more for reading. Refer to Logger memory header list dialog for more information.

  • You can click the toolbar button ([null]) or select the menu item "Tool -> Start Recording" to start recording current measurement values. To stop the recording, click the toolbar button ([null]) or select the menu item "Tool -> Stop Recording". See Recording current measurement values for details.

  • You can click the Go offline toolbar button ([null]) or select the menu item "Device -> Go Offline" to set the device offline.

Notes:

  • You can retrieve recorded measurement values stored in DS 300 when the device is either online or offline. But it is recommended that you do this when the device is offline to speed up the reading.

  • For CS 2390, retrieval of recorded measurement values can be done only when the device is offline. If there is recording going on, it will be stopped before reading starts.
