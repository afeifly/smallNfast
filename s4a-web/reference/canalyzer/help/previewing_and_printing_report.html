Previewing and printing report

CAA allows you to preview and print a report presenting your company information, the graph and the statistics. You can also choose to print the graph only.

Entering basic information

If you want to print the full report, you should enter the basic information about your company in Basic information dialog as this information will appear on the cover page of the printed report. Basic information dialog can be opened by clicking the Basic setting toolbar button () or selecting the menu item "Setting -> Basic Setting". It allows you to enter your company's name, address, phone, etc.



Basic information dialog


  • To set the company logo, click "Select Logo File" button and choose a logo file in the Load dialog. This logo will appear in top right corner of every page of the printed report.



    Load dialog


After clicking "OK" button, all information you entered will be saved and loaded the next time you start the program.

Selecting page printing options

To preview and print the report correctly, page printing options must be properly selected in Page setup dialog. The dialog can be opened by clicking the Page setup toolbar button (), selecting the menu item "File -> Page Setup" or clicking the "Page Setup" button in Print preview dialog.



Page setup dialog (showing default page printing options)


The dialog displays the default/user-defined page printing options for the printer that will print the graph. Here you can choose the paper size you want to use (make sure the printer is loaded with the same paper as you specify here), the page's orientation and its margins. To change the printer for printing the report, click "Printer..." button and choose another printer from the drop-down list. After you click "OK" button, the options you select here will be saved to a file and loaded the next time you start the program. Note: The cover page of the report will always be printed in portrait orientation, regardless of the orientation you select here.

Previewing and Printing report

You can click the Print preview toolbar button () or select the menu item "File -> Print Preview" to preview the report in Print preview dialog.



Print preview dialog


The dialog displays the report on the screen as it would appear when printed. It also indicates the name of the printer that will print the report. To view another page of the report, click "Next" or "Previous" button. In this dialog you can:

  • Select page printing options: click the "Page Setup" button.

  • Print the full report: select "Full report" from View drop-down list and click Print button. The standard Print dialog will appear and you can click "OK" button to print the graph. Please note that you will be prompted to select the page printing options again if you change the printer here. This is to ensure all page printing options are supported by the chosen printer.



    Print dialog


  • Print the graph only: select "Graph only" from View drop-down list and click Print button. The same Print dialog as shown above will appear except that you won't be able to select the print range.

Notes:

  • If the printer you choose to print the report is not loaded with the same paper as you specify in Page setup dialog, the report may be printed incorrectly.

  • When you run a new version of the program for the first time, the page printing options file cannot be found or the computer's default printer has been changed, you will be prompted to select the page printing options in Page setup dialog before the Print preview dialog appears. This is to ensure the report will be correctly displayed and printed.

  • You may click the "Properties" button in the Print dialog to alter the printer's printing options, such as print quality. But you should not alter the page printing options (paper size and orientation) there. The page printing options will only be obtained from the Page setup dialog and changing them again elsewhere will have no effect on the printout.

  • The graph will be printed on white paper, so you should avoid selecting white color for any curve in Select channel dialog.

  • Use a color printer for better printing result for the graph.
