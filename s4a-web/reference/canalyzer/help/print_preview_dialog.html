Print preview dialog


(Missing picture [print preview dialog.jpg])
Print preview dialog


This dialog can be opened by clicking the Print toolbar button ([null]) or selecting the menu item "File -> Print" in graphic view. The dialog displays the graph shown in graphic view on the screen as it would appear when printed. It also indicates the name of the printer that will print the graph. In this dialog you can:

  • Select page printing options: click the "Page Setup" button. The standard Page setup dialog will appear, which displays the default/user-defined page printing options for the printer that will print the graph. Here you can choose the paper size you want to use (make sure the printer is loaded with the same paper as you specify here), the page's orientation and its margins. To change the printer for printing the graph, click "Printer..." button and choose another printer from the drop-down list. After you click "OK" button, the options you select here will be saved to a file and loaded the next time you start the program.


    (Missing picture [page setup dialog.jpg])
    Page setup dialog (showing default page printing options)


  • Print the graph: click the "Print" button. The standard Print dialog will appear and you can click "OK" button to print the graph. Please note that you will be prompted to select the page printing options again if you change the printer here. This is to ensure all page printing options are supported by the chosen printer.


    (Missing picture [print dialog.jpg])
    Print dialog


Notes:

  • If the printer you choose to print the graph is not loaded with the same paper as you specify in Page setup dialog, the graph may be printed incorrectly.

  • When you run a new version of the program for the first time, the page printing options file cannot be found or the computer's default printer has been changed, you will be prompted to select the page printing options in Page setup dialog before the Print preview dialog appears. This is to ensure the graph will be correctly displayed and printed.

  • You may click the "Properties" button in the Print dialog to alter the printer's printing options, such as print quality. But you should not alter the page printing options (paper size and orientation) there. The page printing options will only be obtained from the Page setup dialog and changing them again elsewhere will have no effect on the printout.

  • The graph will be printed on white paper, so you should avoid selecting white color for any curve when setting the view configuration.

  • Use a color printer for better printing result.
