Protocol header dialog


This dialog can be opened by clicking the "Open File" button on main panel, clicking the Open file toolbar button ([null]) or selecting the menu item "File -> Open File". The dialog allows you to view the list of all protocol headers stored in the program's database and provides the following functions:

  • Select a protocol header for viewing: select a protocol header in the list and click "Select" button. The protocol header will be shown in both graphic and table views.

  • View protocol header information: select a protocol header in the list and click "Header Information" button. A dialog will show up displaying the header information.


    (Missing picture [protocol header information dialog.jpg])
    Protocol header information dialog


  • Export protocol header to an Excel file: select a protocol header in the list, click "Export" button and select "To Excel File". Refer to Exporting protocol header for more information.

  • Export protocol headers to a CS Data file: select one or more protocol headers in the list, click "Export" button and select "To CS Data File". Refer to Exporting protocol header for more information.

  • Delete protocol headers: click to select one or more protocol headers in the list and click "Delete Selected Protocol Header" button. You will be asked to confirm the deletion. To Select multiple protocol headers, click while holding the Ctrl key. To select multiple protocol headers in a rows, click the first one and then click the last one while holding the Shift key. Please note that the deletion is final, i.e. after that it is not possible to bring the protocol header back.
