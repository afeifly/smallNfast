Recording current measurement values

You can record current measurement values from CS sensors using the toolbar buttons ([null] and [null]) or the menu items "Tool -> Start Recording" and "Tool -> Stop Recording". The recording can also be started by clicking the "Show Online Value" button in graphic or table view. Note that the recording can be started only when the device is online.

This is the normal scenario for recording current measurement values:

  • When choosing to start the recording, you will be prompted to enter a sample rate in seconds. You must enter an integer value between 5 and 32767. The smaller the sample rate, the more frequently the measurement values are recorded.

  • After the recording starts, you can switch to graphic or table view to view the measurement values as they are recorded. The graph or table in the view will be refreshed at a rate that equals the specified sample rate.

  • When you choose to stop the recording, you will be prompted to enter a filename and save the file to the database or you may cancel saving the file. This file will contain all the recorded measurement values as well as information such as the start time and the end time.

  • After the file is saved, it be will be available as the last item in the list in Protocol header dialog.
