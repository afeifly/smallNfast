Select channel dialog



Select channel dialog


This dialog can be opened by clicking the Select channel toolbar button () or selecting the menu item "Tool -> Select Channel". The dialog allows you to view and configure the channels contained in the currently opened record files. In addition, you can change the Y axes' settings here.

  • The top of the dialog shows the available and selected channels. Only the selected channels' will be visible on the graph, i.e., their measurement values will be represented as curves on the graph. To add a selected channel, select an available channel and click "Select" button. To remove a selected channel, selected a selected channel and click " Remove" button. Please note that the maximum number of different units for the selected channels is two, as the graph can have up to two vertical (Y) axes. There can be up to 6 selected channels as long as the number of different units doesn't exceed two.

  • You can select a selected channel to see the color and style used to draw its curve. Click "Change Color" and "Change Style" buttons to change the color and style respectively. You can choose any color you want from the Choose Line Color dialog. And you can choose either of the two styles: continuous line and dotted line.

  • The two Y axes' settings are shown at the bottom of the dialog. For each Y axis, you can:

    • Change the unit for the scale using the "Unit" drop-down list

    • Enable automatic scaling by checking the box labeled "Automatic Scaling". If you disable this function, you can do the following:

    • Set the scaling by entering the start value in "From" field and the end value in "To" field. The end value should be greater than the start value.

    • Specify the number of steps on the scale by entering a positive integer value in "Number of Steps" field.

The functions of the three buttons at the bottom of this dialog are as follows:

  • Click "OK" button to apply the configuration and close the dialog.

  • Click the "Apply" button to apply the configuration and leave the dialog open.

  • Click the "Cancel" button to discard the modified configuration and close the dialog.
