Select database dialog



Select database dialog


This dialog can be opened by clicking the database toolbar button () or selecting the menu item "File -> Select Database". The dialog displays the list of all available databases and allows you to manage them. These databases can be generated by different versions or copies of CSSoft or CS-Bus. Some databases may be added to the list when program starts up for the first time. In this dialog you can:

  • Select a database for opening: select a database in the list and click the "Select" button. You will be notified whether the opening is successful or not. Once a database has been opened, you can view all records files stored in it in Select record file dialog. The opening will fail if:

    • The database's file path is invalid, e.g. the file in the path is not a valid database file or has been deleted. In this case, you should verify the database's file path.

    • The database is currently used by another program (CSSoft or CS-Bus). In this case, you should close the program using the database and try again.

  • Add a new database to the list: click the "New" button. The Add database connection dialog will appear. You can enter a name and a description for the database.



    Add database connection dialog


    To specify the database file path, click the "Select" button. The Open dialog will appear. You should select the file "csmain.data" in a CSSoft or CS-Bus installation directory (e.g. "C:\Program Files\cssoft").



    Open dialog


    When you return to the Add database connection dialog, just click the "OK" button to add the database to the list.

  • Edit a database: select a database in the list and click the "Edit" button. The Edit database connection dialog will appear, where you can edit the database's name, description and file path.

  • Remove a database: select a databse in the list and click the "Remvoe" button. If the selected database isn't open, it will be removed instantly. If it is open, the program will prompt you for confirmation and remove it only if you answer Yes.
