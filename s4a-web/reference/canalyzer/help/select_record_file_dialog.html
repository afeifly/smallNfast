Select record file dialog



Select record file dialog


This dialog can be opened by clicking the Select file toolbar button () or selecting the menu item "File -> Select File". The dialog allows you to view all record files stored in the currently opened database. If no database is open, no record files will be displayed here. To open a database, go to Select database dialog.

This dialog provides the following functions:

  • Select a record file for viewing: select a record file in the list and click "Select" button. Measurement values stored in the record file will be presented in a graphic form.

  • Select two or more record files for viewing: select two or more record files in the list and click "Compare" button. To Select multiple record files, click while holding the Ctrl key. To select multiple record files in a rows, click the first one and then click the last one while holding the Shift key.

  • View record file information: select a record file in the list and click "File Information" button. A dialog will show up displaying the record file information.



    Record file information dialog


  • Export record files to a CS Data file: select one or more record files in the list and click "Export" button.



    export dialog-c.jpg


  • Delete record files from database: select one or more record files in the list and click "Delete" button. You will be asked to confirm the deletion. Please note that the deletion is final, i.e. after that it is not possible to bring the record files back.
