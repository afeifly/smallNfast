Startup and termination

After having successfully installed CAA, you can double-click the CAA icon () on your Desktop or select the one in your Start menu to start the program.

  • The program will automatically search the default locations (e.g. "C:\Program Files\cssoft") of your computer for CSSoft or CS-Bus databases and configure them if:

    • the program is started for the first time or

    • there were no databases configured when the program was last shut down

    After the search is complete, it will notify you of the number of valid databases found. Also, the program will try to connect to the first database if at least one database has been found. You will see the main window of CAA after pressing the OK button.



    First startup message


  • On other conditions, the program will not search for CSSoft or CS-Bus databases and you will see the main window directly. However, the program will attempt to connect to the database it was accessing when it was last shut down. If this attempt fails, you will be prompted to check the database settings.



    CAA main window


Exiting the program can be done either by the "File -> Exit" menu item or by the standard close window button. In both cases, the program will store the user settings before shutting down.
