Table view


You can switch the program to table view by selecting the menu item "View -> Table View" or clicking the toolbar button ([null]).

The program shows a table containing the retrieved or recorded measurement values in table view. The following features are provided in table view:

  • Each column of the table shows the measurement values for a particular channel. Each row shows the measurement values for all visible channels at a particular time. The table can show measurement values for up to 10 channels. You can change the channels visible in the table in View configuration dialog for table view.

  • The program shows up to 2000 rows of data at a time in a scrollable panel. You can scroll through the table by moving the knob on the vertical or horizontal scroll bar. If there are more than 2000 rows of data, you can click "Next" and "Previous" on top of the table to view another page of the data.

  • As with graphic view, you can click the "Show Online Value" button to start recording and viewing current measurement values. Refer to Show online value for more information.

  • As with table view, you can click the "Open File" button to view all protocol headers stored in the program's database and select one for viewing. Refer to Protocol header dialog for more information.

  • You can click the View configuration toolbar button ([null]) or select the menu item "Configuration -> View" to view and change the view configuration for table view. See View configuration dialog for table view for details.
