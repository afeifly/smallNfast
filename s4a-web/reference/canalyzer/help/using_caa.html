Using CAA

CAA is a graphical application which can be used in Microsoft Windows environment. The program currently provides only English user interface. User interface for other languages will be availabe in a future version of the program.

About the screenshots

As explaining a program has a very important visual component, this documentation features many screenshots of CAA. They have been taken with different versions of the program. Therefore, they might differ from what the program looks like in a current version. However, differences should be small enough so as not to confuse the reader.
