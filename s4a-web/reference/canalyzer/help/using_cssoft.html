Using Compressed Air Analyzer

CSSoft is a graphical application which can be used in Microsoft Windows environment. The program will provide English, German, Chinese and Japanese user interface according to the language setting of your Windows operating system. Whichever user interface it provides, it will function in exactly the same way. The screenshots in this documentation show the program with English user interface.

About the screenshots

As explaining a program has a very important visual component, this documentation features many screenshots of CSSoft. They have been taken with different versions of the program. Therefore, they might differ from what the program looks like in a current version. However, differences should be small enough so as not to confuse the reader.
