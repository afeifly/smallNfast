View configuration dialog (graphic view)


This dialog can be opened by clicking the View configuration toolbar button ([null]) or selecting the menu item "Configuration -> View". The dialog allows you to view and change the view configuration for graphic view.

  • The top of the dialog shows the available and selected channels. Only the selected channels' will be visible on the graph, i.e., their measurement values will be represented as curves on the graph. To add a selected channel, select an available channel and click "Select -->" button. To remove a selected channel, selected a selected channel and click "<-- Remove" button. Please note that the maximum number of different units for the selected channels is two, as the graph can have up to two vertical (Y) axes. There can be up to 10 selected channels as long as the number of different units doesn't exceed two.

  • You can select a selected channel to see the color and style used to draw its curve. Click "Change Color" and "Change Style" buttons to change the color and style respectively. You can choose any color you want from the Choose Line Color dialog. And you can choose either of the two styles: continuous line and dotted line.

  • The title for the graph is shown in "View Header" field and you may change it.

  • The two Y axes' settings are shown at the bottom of the dialog. The right Y axis is enabled, and therefore visible on the graph, only when there are two or more selected channels. You may manually disable it by checking the box labeled "Disable Right Axis". For each Y axis, you can:

    • Change the unit for the scale using the "Unit" drop-down list

    • Enable automatic scaling by checking the box labeled "Automatic Scaling". If you disable this function, you can do the following:

    • Set the scaling by entering the start value in "From" field and the end value in "To" field. The end value should be greater than the start value.

    • Specify the number of steps on the scale by entering a positive integer value in "Number of Steps" field.

The functions of the three buttons at the bottom of this dialog are as follows:

  • Click "OK" button to apply the configuration and close the dialog.

  • Click the "Apply" button to apply the configuration and leave the dialog open.

  • Click the "Cancel" button to discard the modified configuration and close the dialog.

View configuration dialog (table view)


This dialog can be opened by clicking the View configuration toolbar button ([null]) or selecting the menu item "Configuration -> View". The dialog allows you to view and change the view configuration for table view.

  • The top of the dialog shows the available and selected channels. Only the selected channels' will be visible in the table, i.e., their measurement values will be shown in the table. To add a selected channel, select an available channel and click "Select -->" button. To remove a selected channel, selected a selected channel and click "<-- Remove" button. There can be up to 10 selected channels.

  • The title for the table is shown in "View Header" field and you may change it.

The three buttons at the bottom of this dialog have the same functions as those in the dialog for graphic view. See dialog buttons for details.
